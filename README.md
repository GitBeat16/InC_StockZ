📊 InC_StockZ
A stock market simulation and analysis platform designed to help users learn, practice, and experiment with stock trading using virtual money.
🚀 Overview
InC_StockZ is a mock trading application that allows users to:
Track real-time stock data 📈
Simulate buying and selling stocks 💰
Analyze market trends 📊
Practice trading strategies without financial risk
This project is ideal for:
Students preparing for finance/tech exams
Beginners learning stock markets
Hackathon/demo projects
✨ Features
📡 Live Stock Data Integration (via APIs like yfinance)
💼 Virtual Portfolio Management
📉 Buy/Sell Simulation
📊 Performance Tracking
🧠 Risk-Free Learning Environment
🌐 Interactive UI (Streamlit-based)
🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Data Source: yFinance API
Version Control: Git + GitHub
📂 Project Structure
InC_StockZ/
│── app.py                # Main application file
│── modules/             # Custom modules (data handling, logic)
│── requirements.txt     # Dependencies
│── README.md            # Project documentation
⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/GitBeat16/InC_StockZ.git
cd InC_StockZ
2️⃣ Install dependencies
pip install -r requirements.txt
3️⃣ Run the application
streamlit run app.py
🧑‍💻 Usage Guide
Enter a stock ticker (e.g., AAPL, TSLA)
View real-time stock data
Buy stocks using virtual money
Track your portfolio performance
Experiment with strategies
